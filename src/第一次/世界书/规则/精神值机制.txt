  # 精神值机制

  ## 基础定义
  精神值代表角色的心理状态和精力水平，取值范围 0~100。该数值直接影响角色的社交能力、学习效率、判断力和整体叙事氛围。

  ## 被动恢复
  | 状态 | 每小时恢复 |
  |------|----------|
  | 无所事事/放松（散步、发呆、看风景） | +3~6 |
  | 睡眠 | +5~8 |
  | 娱乐活动（看剧、听音乐、打游戏） | +5~10 |

  ## 活动消耗
  | 活动类型 | 消耗量 |
  |---------|--------|
  | 上课（每节课 45~90 分钟） | -4~7 |
  | 自习/写作业/写论文（每小时） | -5~10 |
  | 社团活动（每次） | -3~6 |
  | 考试（每场） | -8~15 |

  ## 社交消耗与恢复
  | 社交类型 | 变化量 |
  |---------|--------|
  | 愉快聊天/朋友相聚 | +5~12 |
  | 尴尬对话/冷场 | -3~7 |
  | 激烈争吵/社交冲突 | -8~15 |
  | 接受帮助/被安慰 | +6~12 |
  | 被批评/当众出丑 | -10~15 |
  | 运动/散步/健身 | +3~8 |

  ## 饱食度联动
  - 饱食度 < 15 时，精神值额外每小时消耗 2~4
  - 饱食度 > 80 且刚进食完，精神值消耗 +1（犯困）

  ## 精神值对叙事的影响

  | 精神值 | 阶段 | 叙事要求 |
  |--------|------|---------|
  | 0~15 | 精神崩溃 | 无法正常对话，回应简短甚至答非所问。学习能力丧失，可能出现翘课、拒绝社交、直接回宿舍的行为。角色可能会不由自主地叹息、趴在桌上。 |
  | 16~35 | 极度疲惫 | 反应明显变慢，上课走神甚至打瞌睡，社交时语气冷淡敷衍。做决定时倾向于最省力的选项。黑眼圈、揉太阳穴等细节应频繁出现。 |
  | 36~55 | 疲惫 | 缺乏积极主动性，能不做的事就不做。上课勉强能听但效率低。社交态度中性偏冷淡，幽默感下降。偶尔叹气或看手机摸鱼。 |
  | 56~75 | 正常 | 可以正常学习、社交、参加活动。思维清晰，对话流畅。 |
  | 76~90 | 良好 | 积极乐观，学习效率高，社交主动。说话有活力，容易开玩笑。愿意尝试新事物或帮助他人。 |
  | 91~100 | 精力充沛 | 元气满满，语速偏快，思维跳脱。什么都愿意试一试，可能会主动组织活动。幽默感强，容易感染周围人的情绪。 |
