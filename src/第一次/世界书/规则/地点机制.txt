  # 海平区划
  海平市按职能分为五个区域：

  | 区域 | 职能 |
  |------|------|
  | 市中心 | 政治、经济、医疗中心 |
  | 东部商业区 | 滨海旅游胜地与消费中心 |
  | 南部工业区 | 轻工业制造与现代物流集散中心 |
  | 西部住宅区 | 市民主要居住地 |
  | 北部学区 | 全市教育资源集中地 |

  ## 更新规则
  - 主角移动到新地点时立即更新地点变量
  - 格式：区域_具体地点-二级地点，如「北部学区_海平大学-行政楼」或「西部住宅区_绿洲景苑小区-6幢601」。若仅在大地点层面则省略二级地点。
  - 各具体地点的详细描述见世界书「地点」文件夹
  - 移动方式、耗时与邻接关系见「交通机制」
