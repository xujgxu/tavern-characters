  # 时间推进机制

  ## 基础定义
  时间格式为 YYYY-MM-DD HH:MM。每一次 AI 回复都应推理当前时间并更新。

  ## 时间推进规则
  - 单次 AI 回复覆盖的游戏时间：对话交互 3~15 分钟 / 上课 45~90 分钟 / 吃饭 20~40 分钟
  - 累计时间超过 2 小时时应考虑是否自然结束当前场景
  - 睡眠行为直接跳转至次日早晨 7:00~9:00

  ## 行为时间映射

  | 行为 | 消耗时间 |
  |------|----------|
  | 短对话（打招呼、闲聊几句） | 3~8 分钟 |
  | 中对话（深入交谈、商量事情） | 10~20 分钟 |
  | 步行同区域 | 3~5 分钟 |
  | 步行跨区域（如教学楼到食堂） | 10~20 分钟 |
  | 上课（一节） | 45~90 分钟 |
  | 课间休息 | 10~15 分钟 |
  | 食堂吃饭 | 20~30 分钟 |
  | 外卖/外出就餐 | 30~50 分钟 |
  | 洗澡 | 15~25 分钟 |
  | 自习/图书馆 | 60~120 分钟 |
  | 午睡 | 30~90 分钟 |
  | 社团活动 | 60~120 分钟 |
  | 运动/健身 | 45~90 分钟 |
  | 购物逛街 | 30~90 分钟 |
  | 娱乐（看一集剧/打一局游戏） | 20~60 分钟 |
  | 睡眠 | 直接推进至次日早上 |

  ## 一天时段参考
  | 时段 | 时间范围 | 通常活动 |
  |------|----------|---------|
  | 早晨 | 06:00-08:30 | 起床、洗漱、早餐 |
  | 上午 | 08:30-12:00 | 上课或自习 |
  | 午间 | 12:00-14:00 | 午餐、午休 |
  | 下午 | 14:00-17:30 | 上课或活动 |
  | 傍晚 | 17:30-19:00 | 晚餐 |
  | 晚间 | 19:00-23:00 | 自习/社团/娱乐 |
  | 深夜 | 23:00-06:00 | 睡眠 |

  ## 特殊规则
  - 周二/周四上午没课（根据第一次消息设定），角色可以睡懒觉或自由安排
  - 周末时间安排更加自由，无课程约束
  - 期末考试周学习压力增大，自由时间减少
