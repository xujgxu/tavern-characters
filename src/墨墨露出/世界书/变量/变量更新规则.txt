#变量更新规则
在所有文本的最后，进行变量更新。

rule:
  - You must output the update analysis in the end of the next reply
  - Only update variables when their specific update conditions are met
  - Do not repeatedly update the same variable for the same event in continuous discussion
  - Every update must include a clear and brief reason related to current interaction

---
变量更新规则:
  世界:
    当前时间:
      format: YYYY-MM-DD HH:MM
      check:
        - 根据回复正文中描述的动作和事件，模拟现实中完成这些事情所需的时间来推进
        - 参考锚点（单次动作的时间消耗）：
          - 打开App/发一条消息/穿脱一件衣服：不超过1分钟
          - 刷牙洗脸/更换整套服装：3-5分钟
          - 吃一顿饭/洗完澡：15-30分钟
          - 步行一个公交站/坐地铁2-3站：10-15分钟
          - 等公交/地铁：2-10分钟
          - 上课一节课：45分钟
          - 实习半日：4小时
          - 跳过时间段（如"过了20分钟"）：按描述中的时间推进
        - 多条回复中的连续动作累积时间应合理叠加，不应每次独立重置
    当前地点:
      format: "区域_具体位置"
      check:
        - 移动到新位置时更新
    近期事务:
      type: |-
        {
          [事务名: string]: string; // 事务描述
        }
      check:
        - 记录需要完成的任务、约定、重要事件等
        - 完成后从列表中移除
  主角:
    姓名:
      check:
        - 初始设定即可，通常不需修改
    年龄:
      type: number
      check:
        - 生日时更新
    身份:
      check:
        - 工作、角色身份变化时更新
    金钱:
      type: number
      check:
        - 实习工资、任务报酬、奖励等收入时增加
        - 消费、购物、还款、交租时减少
        - 不得为负数
    衣物:
      type: |-
        {
          [服装部位: string]: string; // 服装名称+描述
        }
      check:
        - 换装时更新对应部位
  数值:

      type: number
      range: 0~100
      check:
        - 随时间自然消耗（睡眠-1/h，静息-2/h，活动-4/h，剧烈运动-10/h）
        - 进食增加（零食+15，正餐+40）
    精神值:
      type: number
      range: 0~100
      check:
        - 放空状态-1/h，社交-2/h，高强度脑力-6/h，喝酒-25/杯
        - 高质量睡眠+12/h，打盹+5/h，功能饮料+20/次
        - 归零时强制昏睡12小时，醒来仅恢复至50
    心情值:
      type: number
      range: 0~100
      check:
        - 自然流失-1/h
        - 娱乐、高级消费、正面事件增加
        - 被指责、负面事件减少
        - 归零时游戏结束
    性欲值:
      type: number
      range: 0~100
      check:
        - 自然积累+1/h，性刺激+1/次，性行为中+5/分钟
        - 高潮后清零
    快感值:
      type: number
      range: 0~100
      check:
        - 轻度性刺激+5/分钟，深度性刺激+10/分钟
        - 刺激停止-20/分钟，高潮后重置为60
    整体色情值:
      type: number
      range: 0~100
      check:
        - 根据当前穿着服装的色情值平均值动态计算
        - 遵循视觉掩蔽原则和风格标签修正
  日程:

      check:
        - 每天随着时间推进更新
    星期:
      check:

    是否学期:
      type: boolean

        - 根据年度时间表切换
    固定日程:
      type: |-
        [
          {
            时间: string;

            额外要求?: string;

      type: |-
        {
          任务名: string;

          - 已经过期的事件在分析阶段标记完成
